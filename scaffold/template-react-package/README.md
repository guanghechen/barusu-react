# Usage

  * Install
    ```shell
    npm install -g @barusu-react/template-react-package

    # or use yarn:
    # yarn global add @barusu-react/template-react-package
    ```

  * Create a typescript+stylus+react project
    ```shell
    create-react-package [package name]
    ```
