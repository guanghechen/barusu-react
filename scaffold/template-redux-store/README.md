# Usage

  * Install
    ```shell
    npm install -g @barusu-react/template-redux-store

    # or use yarn:
    # yarn global add @barusu-react/template-redux-store
    ```

  * Create a redux store
    ```shell
    create-redux-store new:store
    ```

  * Create a store state node
    ```shell
    create-redux-store new:state
    ```
